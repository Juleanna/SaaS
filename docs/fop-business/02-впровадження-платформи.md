## 2. Розгортання на продакшені

### Крок 1: Підготовка сервера

```bash
# Оновити систему
sudo apt update && sudo apt upgrade -y

# Встановити Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# Встановити Docker Compose
sudo apt install docker-compose-plugin -y
```

### Крок 2: Налаштування домену

1. Зареєструвати домен (наприклад, на nic.ua — від 200 грн/рік для .ua)
2. Додати DNS-записи:
   - `A` запис: `yourdomain.com` → IP сервера
   - `A` запис: `*.yourdomain.com` → IP сервера (для піддоменів магазинів)
   - `CNAME` запис: `www` → `yourdomain.com`

### Крок 3: SSL-сертифікат

```bash
# Встановити Certbot
sudo apt install certbot -y

# Отримати wildcard-сертифікат (для піддоменів)
sudo certbot certonly --manual --preferred-challenges=dns \
  -d "yourdomain.com" -d "*.yourdomain.com"
```

Або використати Cloudflare (безкоштовний SSL + CDN + DDoS-захист).

### Крок 4: Деплой через Docker Compose

```bash
# Клонувати проект
git clone <repo-url> /opt/saas
cd /opt/saas

# Створити .env для продакшену
cp backend/.env backend/.env.prod

# Відредагувати .env.prod:
# - SECRET_KEY=<згенерувати новий>
# - DEBUG=False
# - ALLOWED_HOSTS=yourdomain.com,*.yourdomain.com
# - DB_PASSWORD=<надійний пароль>

# Запустити через production compose
docker compose -f docker-compose.prod.yml up -d --build
```

### Крок 5: Налаштування Nginx (reverse proxy)

Якщо не використовується prod compose з nginx, налаштувати окремо:

```nginx
server {
    listen 80;
    server_name yourdomain.com *.yourdomain.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl;
    server_name yourdomain.com *.yourdomain.com;

    ssl_certificate /etc/letsencrypt/live/yourdomain.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/yourdomain.com/privkey.pem;

    location /api/ {
        proxy_pass http://localhost:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 3. Інтеграція платіжних систем для України

### LiqPay (ПриватБанк) — рекомендовано для старту
- **Комісія:** 2.75% за транзакцію
- **Підключення:** потрібен рахунок у ПриватБанку
- **API:** REST API + SDK для Python
- **Підтримує:** Visa, MasterCard, Google Pay, Apple Pay

```python
# Приклад інтеграції LiqPay
# pip install liqpay-python
from liqpay import LiqPay

liqpay = LiqPay(PUBLIC_KEY, PRIVATE_KEY)
html = liqpay.cnb_form({
    'action': 'pay',
    'amount': '100',
    'currency': 'UAH',
    'description': 'Замовлення #123',
    'order_id': 'order_123',
    'server_url': 'https://yourdomain.com/api/payments/callback/',
})
```

### Fondy (monobank Acquiring)
- **Комісія:** від 2% за транзакцію
- **Підключення:** онлайн через fondy.ua
- **Переваги:** швидке підключення, підтримка підписок

### WayForPay
- **Комісія:** від 2.5% за транзакцію
- **Підключення:** онлайн через wayforpay.com
- **Переваги:** популярний в Україні, фіскалізація

### Stripe (для міжнародних платежів)
- **Комісія:** 2.9% + $0.30
- **Примітка:** Stripe доступний в Україні з обмеженнями, підходить для роботи з іноземними клієнтами

## 5. Юридичні вимоги до SaaS в Україні

### Обов'язкові документи на сайті
1. **Договір публічної оферти** — умови використання платформи
2. **Політика конфіденційності** — обробка персональних даних (відповідно до ЗУ "Про захист персональних даних")
3. **Умови повернення коштів** — політика рефандів
4. **Інформація про ФОП** — назва, ІПН, адреса (обов'язково для інтернет-торгівлі)

### Захист персональних даних
- Зареєструвати базу персональних даних у Уповноваженого ВРУ з прав людини (якщо обробляєте дані клієнтів)
- Отримувати згоду користувачів на обробку даних (чекбокс при реєстрації)
- Зберігати дані на серверах з адекватним захистом (шифрування, контроль доступу)
