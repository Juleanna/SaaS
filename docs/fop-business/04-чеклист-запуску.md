# Чеклист запуску SaaS-платформи

## Юридична підготовка
- [ ] Зареєструвати ФОП (через Дію або ЦНАП)
- [ ] Обрати 3 групу єдиного податку (5%)
- [ ] Зареєструвати КВЕДи (63.11, 62.01, 62.09, 63.12)
- [ ] Відкрити банківський рахунок ФОП
- [ ] Отримати КЕП (кваліфікований електронний підпис)
- [ ] Зареєструватися в кабінеті платника ДПС

## Юридичні документи для сайту
- [x] Підготувати Договір публічної оферти → [06-договір-публічної-оферти.md](06-договір-публічної-оферти.md)
- [x] Підготувати Політику конфіденційності → [07-політика-конфіденційності.md](07-політика-конфіденційності.md)
- [x] Підготувати Умови повернення коштів → [08-умови-повернення-коштів.md](08-умови-повернення-коштів.md)
- [ ] Заповнити реквізити ФОП у всіх документах (ПІБ, ІПН, адреса, IBAN, контакти)
- [ ] Розмістити документи на сайті (footer або окремі сторінки)
- [ ] Додати чекбокс згоди при реєстрації: "Ознайомлений з Офертою та Політикою конфіденційності"

## Технічна підготовка
- [ ] Орендувати VPS-сервер
- [ ] Зареєструвати домен
- [ ] Налаштувати DNS (A-записи, wildcard)
- [ ] Встановити Docker на сервер
- [ ] Налаштувати SSL-сертифікат
- [ ] Розгорнути проект через docker-compose.prod.yml
- [ ] Налаштувати Nginx reverse proxy
- [ ] Налаштувати автоматичні бекапи БД
- [ ] Налаштувати моніторинг (UptimeRobot)

## Платіжна система
- [ ] Підключити LiqPay (або Fondy/WayForPay)
- [ ] Протестувати тестові платежі
- [ ] Налаштувати webhook для підтвердження оплат
- [ ] Протестувати процес повернення коштів

## Перед запуском
- [ ] Провести тестування всіх функцій
- [ ] Перевірити мобільну адаптацію
- [ ] Змінити SECRET_KEY на продакшені
- [ ] Вимкнути DEBUG=False
- [ ] Створити першого тестового продавця та магазин
- [ ] Оформити тестове замовлення від початку до кінця
- [ ] Підготувати лендінг-сторінку платформи

## Запуск
- [ ] Запустити рекламу в Instagram/Facebook
- [ ] Опублікувати пости в тематичних спільнотах
- [ ] Запросити перших 10 продавців безкоштовно (бета-тестери)
- [ ] Збирати зворотний зв'язок та покращувати
