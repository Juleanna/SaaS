## 🚀 Швидкий старт

### 1️⃣ Реєстрація

```
🌐 http://localhost:5173/register

1. Натиснути "Реєстрація"
2. Заповнити форму:
   ├─ Ім'я користувача (без пробілів)
   ├─ Email (дійсна адреса)
   ├─ Ім'я та прізвище
   ├─ Назва магазину (опціонально)
   └─ Пароль (мінімум 8 символів)
3. Прийняти умови користування ✓
4. Натиснути "Створити акаунт"
```

### 2️⃣ Вхід у систему

```
🌐 http://localhost:5173/login

1. Натиснути "Вхід"
2. Ввести ім'я користувача та пароль
3. Опціонально: Позначити "Запам'ятати мене" ✓
4. Натиснути "Увійти"

💡 Тест: Демо ім'я: demo | Пароль: demo1234
```

### 3️⃣ Перший запуск

```
📊 http://localhost:5173/dashboard

Ви побачите:
✓ Статистика (замовлення, доходи)
✓ Ваші магазини
✓ Останні замовлення
✓ Кнопка для створення магазину
```

## 👤 Профіль користувача

### Як відкрити профіль?

```
1. Натиснути на аватар у правому куті (вгорі)
2. Вибрати "👤 Мій профіль"
```

### Що можна змінити?

| Поле       | Описання           | Обов'язково |
| ---------- | ------------------ | ----------- |
| 👤 Ім'я    | Ваше ім'я          | ✓           |
| 📧 Email   | Адреса для зв'язку | ✓           |
| 📱 Телефон | Номер телефону     | ✗           |
| 📷 Аватар  | Фото профілю       | ✗           |
| 🔒 Пароль  | Безпека акаунту    | -           |

### 🔒 Зміна пароля

```
1. Перейти на сторінку профілю
2. Натиснути "Змінити пароль"
3. Ввести старий пароль
4. Ввести новий пароль (2 рази)
5. Натиснути "Зберегти"
```

## 🏪 Управління магазинами

### 📍 Де знаходиться?

```
🌐 http://localhost:5173/dashboard

Або скористатися меню:
Navbar → Панель управління → (прокрутити вниз)
```

### ➕ Створити новий магазин

```
1. Натиснути "+ Новий магазин"
2. Заповнити форму:
   ├─ 📛 Назва магазину (видна для покупців)
   ├─ 🔗 URL slug (приклад: my-awesome-shop)
   ├─ 📝 Описання (про ваш магазин)
   ├─ 🎨 Категорія (Одяг, Електроніка, тощо)
   ├─ 📱 Телефон (для контактів)
   ├─ 📧 Email (для замовлень)
   └─ 🏙️ Місто
3. Натиснути "Створити магазин"
```

### ✏️ Редагування магазину

```
1. На панелі управління натиснути картку магазину
2. Натиснути "Редагувати"
3. Змінити потрібну інформацію
4. Натиснути "Зберегти"
```

### 🎨 Кастомізація магазину

```
1. У налаштуваннях магазину натиснути "Оформлення"
2. Змінити:
   ├─ 🎨 Основний колір (#FF6B6B)
   ├─ 🖼️ Вторинний колір (#4ECDC4)
   ├─ 📷 Логотип магазину
   ├─ 🖼️ Банер магазину
   └─ 📝 Приватна безпека
3. Натиснути "Зберегти оформлення"
```

### 🌐 Публічна сторінка магазину

```
Посилання: http://localhost:5173/store/{slug}

Приклад: http://localhost:5173/store/my-awesome-shop

Тут покупці можуть:
✓ Переглядати товари
✓ Додавати в корзину
✓ Виконувати покупки
✓ Залишати коментарі
```

### 🗑️ Видалити магазин

```
⚠️ УВАГА: Це видалить весь контент!

1. Натиснути "Налаштування магазину"
2. Прокрутити вниз
3. Натиснути "Видалити магазин"
4. Підтвердити видалення (введіть назву)
```

## 🛍️ Управління товарами

### 📍 Де знаходиться?

```
🌐 http://localhost:5173/products

Або: Navbar → Товари
```

### ➕ Додати товар

```
1. Натиснути "+ Новий товар"
2. Заповнити форму:
   ├─ 📛 Назва товара
   ├─ 📝 Описання (до 500 символів)
   ├─ 💰 Ціна (у гривнях)
   ├─ 📦 Кількість на складі
   ├─ 🏷️ Категорія
   ├─ 📷 Зображення товара
   ├─ 🏷️ SKU (код товара)
   └─ 📏 Розміри (для одягу)
3. Натиснути "Створити товар"
```

### ✏️ Редагування товара

```
1. Натиснути на товар у списку
2. Натиснути "Редагувати"
3. Змінити необхідні поля
4. Натиснути "Зберегти"
```

### 🖼️ Завантаження фотографій

```
1. У формі товара натиснути область для завантаження
2. Вибрати файл (JPG, PNG, WebP до 5MB)
3. Зображення завантажиться автоматично
4. Можна додати до 5 фото на товар
```

### 🔍 Пошук товарів

```
1. У верхній частині натиснути поле пошуку
2. Ввести назву або SKU товара
3. Результати оновлюються автоматично
4. Натиснути на товар для деталей
```

### 📊 Статистика товара

```
Кожен товар показує:
✓ 👁️ Перегляди
✓ 🛒 Додано в корзину
✓ ✅ Куплено
✓ ⭐ Рейтинг
✓ 💬 Коментарі
```

### 🗑️ Видалити товар

```
1. Натиснути на товар
2. Натиснути "Видалити"
3. Підтвердити видалення
```

## 📦 Замовлення

### 📍 Де знаходиться?

```
🌐 http://localhost:5173/orders

Або: Navbar → Замовлення
```

### 👁️ Переглядати замовлення

```
Таблиця показує:
├─ 🔢 Номер замовлення
├─ 👤 Ім'я покупця
├─ 📦 Кількість товарів
├─ 💰 Сума замовлення
├─ 📍 Статус
└─ 📅 Дата замовлення
```

### 🔍 Фільтрація замовлень

```
1. Натиснути на фільтри вгорі
2. Вибрати:
   ├─ Статус (очікує, обробляється, відправлено)
   ├─ Діапазон дат
   └─ Метод оплати
3. Натиснути "Застосувати"
```

### 📋 Деталі замовлення

```
1. Натиснути на замовлення у списку
2. Модальне вікно покажеь:
   ├─ 👤 Інформація про покупця
   ├─ 📮 Адреса доставки
   ├─ 🛍️ Список товарів з цінами
   ├─ 💰 Сума, податок, доставка
   ├─ 💳 Метод оплати
   └─ 📝 Нотатки до замовлення
```

### 🔄 Змінити статус замовлення

```
1. Відкрити деталі замовлення
2. У полі "Статус" вибрати новий статус:
   ├─ ⏳ Очікує - нове замовлення
   ├─ ⚙️ Обробляється - готується до відправлення
   ├─ 🚚 Відправлено - відправлено покупцю
   ├─ ✅ Доставлено - доставлено
   ├─ ❌ Скасовано - відмінено покупцем
   └─ 💵 Повернено - повернуті гроші
3. Натиснути "Зберегти"
4. Покупець отримає уведомлення
```

### 📧 Відправити повідомлення

```
1. Відкрити замовлення
2. Натиснути "📧 Відправити повідомлення"
3. Написати повідомлення
4. Натиснути "Відправити"
5. Буде відправлено email покупцю
```

### 🔐 Дані про платіж

```
Замовлення показує:
✓ 💳 Тип платежу (Stripe/PayPal/ЮKassa)
✓ 💰 Сума платежу
✓ ✅ Статус оплати (оплачено/очікує)
✓ 📅 Дата платежу
✓ 📋 ID транзакції
```

## 💳 Платіжні системи

### 📍 Налаштування платежів

```
🌐 http://localhost:5173/settings/payments

Або: Navbar → Профіль → ⚙️ Налаштування → Платежі
```

### 💳 Stripe

#### ✅ Підключення

```
1. Натиснути "Підключити Stripe"
2. Ввести данные:
   ├─ Публічний ключ (pk_...)
   └─ Секретний ключ (sk_...)
3. Натиснути "Зберегти"
4. Тестування: використовуйте карту 4242 4242 4242 4242
```

#### 🧪 Тестові карти

```
✅ Успішна оплата:
   Номер: 4242 4242 4242 4242
   Дата: 12/25
   CVC: 123

❌ Відхилена оплата:
   Номер: 4000 0000 0000 0002
   Дата: 12/25
   CVC: 123
```

### 🏦 PayPal

#### ✅ Підключення

```
1. Натиснути "Підключити PayPal"
2. Ввести данные:
   ├─ Client ID
   └─ Client Secret
3. Натиснути "Зберегти"
```

### 🇷🇺 ЮKassa (для РФ та СНД)

#### ✅ Підключення

```
1. Натиснути "Підключити ЮKassa"
2. Ввести данные:
   ├─ Shop ID
   └─ Secret Key
3. Натиснути "Зберегти"
```

### 🧪 Тестування платежів

```
1. Як покупець, перейти на публічну сторінку магазину
2. Додати товар у корзину
3. Натиснути "Оформити замовлення"
4. Ввести дані:
   ├─ Ім'я та прізвище
   ├─ Email
   ├─ Телефон
   └─ Адреса доставки
5. Вибрати платіжну систему
6. Виконати платіж
7. Перевірити замовлення на панелі управління
```

## 💬 Telegram Бот

### 📍 Як включити Telegram бот?

```
1. Перейти на http://localhost:5173/settings/telegram
2. Натиснути "Підключити Telegram"
3. Скопіювати ссилку на бота
4. Поділитися посиланням з покупцями
```

### 🤖 Що может робити бот?

```
✓ 📦 Показувати товари з магазину
✓ 🛒 Додавати в корзину
✓ 💬 Показувати ціни
✓ 🔗 Посилання на ваш магазин
✓ 💾 Зберігати пошук користувача
```

### 🚀 Користування ботом (для покупців)

```
1. Відкрити Telegram
2. Знайти бот за посиланням з вашого магазину
3. Натиснути /start
4. Вибрати магазин
5. Переглядати товари
6. Натиснути "Додати в корзину"
7. Натиснути "Оформити замовлення"
8. Перейти на сайт для оплати
```

### 📱 Команди бота

```
/start        - Головне меню
/help         - Довідка по командам
/stores       - Список магазинів
/cart         - Моя корзина
/account      - Налаштування акаунту
```

```

```

   ├─ 📦 Товар для постингу
